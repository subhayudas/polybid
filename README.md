# Polybid - Vendors Portal

A modern web portal for vendors to manage their applications, orders, and business relationship with Polyform Print Studio.

## Features

- Vendor application management
- Order tracking and management
- Dashboard with analytics
- Profile and settings management
- Real-time notifications

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Supabase (for backend)
- React Router (for routing)

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or bun

### Installation

```bash
npm install
# or
bun install
```

### Development

```bash
npm run dev
# or
bun run dev
```

### Build

```bash
npm run build
# or
bun run build
```
